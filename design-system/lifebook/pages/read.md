# Read Page Overrides

> **PROJECT:** Lifebook
> **Page Type:** Reader / Long-form

> ⚠️ **IMPORTANT:** Rules in this file **override** the `MASTER.md` file in this same project folder.
> Only deviations from the Master are documented here. For all other rules, refer to the Master.

---

## Page-Specific Rules

### Typography Overrides

- The read page keeps dedicated reader controls for font family, theme, size, and spacing.
- Default reader spacing should map to the shared baseline reading line-height: `1.72`.
- Segment body content should use `strong reading text` (`95%` foreground opacity).
- Supporting long-form blocks such as the big idea should use `default reading text` (`92%` foreground opacity).

### Interaction Overrides

- Reader controls may override the baseline spacing, but they must remain anchored to the shared reading token system.
- Do not use ad hoc muted gray values for primary reading content.
